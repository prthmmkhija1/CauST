Installation
============

Requirements
------------

- Python >= 3.9
- PyTorch >= 1.12
- PyTorch Geometric
- scanpy, anndata, scikit-learn, scipy, matplotlib

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/prathammakhija/CauST.git
   cd CauST
   pip install -e .

Optional: STAGATE / GraphST
----------------------------

.. code-block:: bash

   pip install STAGATE-pyG
   pip install graphst
