# tutorials/output — Contents

This folder holds outputs generated by the tutorial notebooks.

## causal_gene_scores.csv
Generated by `tutorials/01_quickstart.ipynb`.

The gene names here (`gene_7`, `gene_10`, etc.) are **synthetic** — the
quickstart notebook builds a random 100-gene × 500-spot dataset so that
the tutorial can run instantly without downloading any data. The synthetic
data intentionally has 10 "causal" genes and 10 "confounding" genes so the
reader can verify that CauST correctly recovers them.

For real DLPFC gene names (PENK, HOXC5, ADAM11, AC005332.8, …) see:
- `experiments/results/multi_slice/invariance_scores.json` — top invariant
  genes from the 4-slice multi-donor DLPFC run
- `tutorials/05_causal_gene_exploration.ipynb` — explores these real gene
  scores interactively

## cross_slice_heatmap.png
Generated by `tutorials/04_cross_slice_evaluation.ipynb`.
Shows the cross-slice invariance heatmap for top-50 genes across the 4
DLPFC slices used in the multi-slice run (real data, real gene names).
